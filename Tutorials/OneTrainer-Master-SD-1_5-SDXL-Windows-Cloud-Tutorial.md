### How To Update OneTrainer To The Latest Version

* Open a terminal 
* Execute below to move into accurate folder
* ```cd apps/OneTrainer```
* Execute below to activate VENV of OneTrainer
* ```conda activate ot```
* Execute below to update OneTrainer to the latest version
* ```./update.sh```
